## 5. ABOUT THE PROJECT

HireHub is a modern job portal web application designed to revolutionize the recruitment process. The platform serves two primary user groups:

**For Job Seekers:**
- Intelligent job discovery with multi-criteria filters
- AI Resume Builder with ATS-friendly formatting
- AI Match Analysis showing compatibility with job postings
- Visual application tracking with status pipeline
- Interactive skill assessment quizzes
- Real-time messaging with recruiters

**For Employers:**
- Comprehensive recruiter dashboard with analytics
- Kanban-style applicant tracking
- Custom quiz creation for screening
- Company branding customization
- Interview scheduling management

The application uses Spring Boot for the backend API and React with Vite for the frontend, with real-time features powered by WebSocket.
